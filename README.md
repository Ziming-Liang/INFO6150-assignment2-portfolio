# Portfolio Website - Ziming Liang

## HTML Tags Used

- `<header>` - Page header
- `<nav>` - Navigation menu
- `<section>` - Content sections
- `<article>` - About content
- `<aside>` - Sidebar
- `<footer>` - Page footer
- `<h1>`, `<h2>`, `<h3>` - Headings
- `<p>` - Paragraphs
- `<a>` - Links
- `<div>` - Containers
- `<table>` - Skills table
- `<tr>` - Table rows
- `<th>` - Table headers
- `<td>` - Table cells
- `<form>` - Contact form
- `<label>` - Form labels
- `<input>` - Text inputs
- `<textarea>` - Message input
- `<button>` - Submit button
- `<img>` - Images

## CSS Features

- Flexbox layout with all required properties
- Table styling with multiple selectors
- Hover effects on testimonials and gallery
- Responsive design for iPad (768px) and iPhone (375px)
